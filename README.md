# 🤖 LLM Prompt Benchmarking Framework

Система для автоматизированного A/B тестирования и оценки качества ответов нейросетей (LLM-as-a-Judge).

### 💡 Проблема
При разработке ИИ-агентов сложно понять, какая инструкция (промпт) работает лучше на большой дистанции. Тестировать вручную 100+ вопросов — долго и субъективно.

### 🛠 Решение
Я собрал конвейер, который:
1. **Собирает данные (n8n):** Параллельно прогоняет список вопросов через две разные версии промптов.
2. **Анализирует (Python + Pandas):** Обрабатывает результаты и готовит их к судейству.
3. **Оценивает (LLM-as-a-Judge):** Третья, независимая нейросеть-судья сравнивает ответы по критериям точности и полноты.
4. **Визуализирует:** Генерирует интерактивный HTML-отчет с прокруткой и цветовой индикацией победителей.

### 📈 Результаты
Фреймворк позволяет за 5 минут протестировать сотни сценариев и получить наглядную статистику: какой подход эффективнее и почему (судья пишет вердикт по каждой паре).

### 🚀 Технологический стек
n8n (No-code автоматизация) | Python | Pandas | OpenRouter API | HTML/CSS
